# AURA E-BANK

A Web-Based Digital Banking Management System

## Overview

AURA E-BANK is a web-based digital banking management system that enhances the efficiency and organization of banking operations by:

- **Digitizing traditional banking tasks** to eliminate manual errors and processing delays through automated deposits, withdrawals, fund transfers, and real-time transaction logging.

- **Enhancing visibility of financial activities, user accounts, and banking records** with live balance updates, unique reference numbers per transaction, and fully searchable transaction history.

- **Simplifying core banking functions and delivering precise financial reports** for deposits, withdrawals, fund transfers, bill settlements, loan requests, and comprehensive bank statistics.

## Features

### User (Customer)
- Register with bcrypt password hashing
- Deposit and withdraw funds
- Transfer money to other users (unique reference numbers)
- Pay bills (Electricity, Water, Internet, Mobile, Credit Card)
- Apply for loans (Personal, Home, Car, Business, Education)
- Track loan application status
- View searchable transaction history
- Update own profile (email, phone, password)

### Staff
- Manage user accounts (edit, reset password, freeze/unfreeze)
- Approve or decline loan applications
- View global transaction logs (read-only)

### Admin
- All staff capabilities
- View bank statistics (total capital, total users)
- Undo transactions
- Permanently delete user accounts
